Privacy Policy

Last Updated: July 17, 2026

Universal Command Palette is built with privacy as a core principle. The extension operates locally within your browser and does not collect, sell, rent, or share your personal information.

Information We Access

To perform commands requested by you, the extension may temporarily access the content of the currently active webpage, including text, links, images, tables, and other visible page elements. This access occurs entirely on your device and is used solely to execute the command you selected.

Information We Collect

Universal Command Palette does not collect or transmit:

  • Name
  • Email address
  • Passwords
  • Financial information
  • Authentication credentials
  • Location data
  • Browsing history
  • Personal communications

Local Storage

The extension stores user preferences such as settings and configuration using Chrome's local storage APIs. These settings remain on your device and are never transmitted to us.

Downloads

Some commands allow you to download files, such as images, from webpages. Downloads occur only after you explicitly invoke a download command.

Bookmarks and Tabs

If you use bookmark or tab-related commands, the extension accesses your bookmarks or open tabs locally to perform those commands. This information is never transmitted outside your browser.

Data Sharing

We do not sell, rent, or share user data with third parties.

Analytics

Universal Command Palette does not use analytics, telemetry, tracking pixels, advertising identifiers, or usage monitoring.

Remote Code

The extension does not download or execute remote JavaScript, WebAssembly, or dynamically evaluated code. All executable code is packaged with the extension.

Security

All processing is performed locally within your browser whenever possible.

Changes

This Privacy Policy may be updated to reflect changes to the extension. The "Last Updated" date will always indicate the latest revision.

Contact

For questions regarding this Privacy Policy, please contact the developer through the project's GitHub repository or support page.
